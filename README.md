## Overview

Hyrox grew from 100K participants in 2023 to ~700K by 2026, with most beginners arriving with no idea how to balance running and functional-fitness work. Generic CrossFit programming doesn't prepare them for the specific run/work alternation. Free 12-week plans exist but are PDFs scraped from blog posts with no progression logic.

This product is a structured 12-week plan with two tiers (beginner: 0 → first finish; intermediate: improve sub-90-min target), week-by-week run/strength programming, sled-push/wallball/burpee-broad-jump skill drills, and a race-day pacing calculator.
